## Use

```bash
git clone https://github.com/ResearchForumOnline/FrontDeskAgent-Playbooks.git
cd FrontDeskAgent-Playbooks
python tools/validate_playbooks.py
```

Copy a playbook into a local agent, OpenZero workflow, FrontDeskAgent knowledge base, or another receptionist stack that can follow structured JSON instructions.
